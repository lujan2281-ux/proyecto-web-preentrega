# Proyecto de Desarrollo Web - Pre-Entrega

## Descripción
Este sitio web es una landing page responsiva que simula la estructura de una plataforma de servicios/productos. Incluye navegación interna, secciones ordenadas con Flexbox y Grid, contenido multimedia y un formulario de contacto funcional.

## Tecnologías utilizadas
* HTML5 (Estructura semántica)
* CSS3 (Flexbox, Grid y Media Queries)
* Google Fonts
* Formspree
